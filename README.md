# Invoice Generator

A template-based **Invoice Generator** developed with Java, designed to create
professional and customizable invoices and export them as **PDF files**.

This project focuses on clean invoice layout, flexibility, and easy adaptation
to different business requirements.

> This public repository is intended for **demonstration purposes only**.
> The complete source code is maintained in a private repository.

---

## Project Overview

The Invoice Generator allows users to generate structured and professional
invoices by combining company information, client details, and invoice items
into a well-formatted PDF document.

The system is built around a template-driven architecture, making it easy to
customize invoice appearance without changing core logic.

---

## Key Features

- Dynamic invoice generation with company and client information
- Editable invoice items (quantity, unit price, discounts, taxes)
- Automatic layout balancing for consistent visual appearance
- Summary section including subtotal, tax, discounts, and total amount
- Customizable terms and instructions section
- Template-based design for easy styling and branding
- Support for mock data during development and testing

---

## Technologies & Tools

- Java 8 / Java 11+
- Apache Velocity (template engine)
- Flying Saucer (PDF generation)
- SLF4J (logging)
- Mockoon (mock API testing)

---

## Architecture & Design

- Template-driven invoice rendering using HTML and CSS
- Separation of data, template, and rendering logic
- PDF generation based on web-standard technologies
- Flexible structure that can be extended or integrated into other systems
